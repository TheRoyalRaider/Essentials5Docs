firework

Allows you to modify a stack of fireworks
Usage: /firework {meta} OR /firework {power|fire} {amount} OR /firework clear
This command allows you modify the effects/charges on a stack of fireworks. While holding the stack you'd like to modify, use the command to either clear or add an effect.

Full Meta Syntax:

  • color:{color[,color,..]}
  • fade:{color[,color,..]}
  • shape:{star|ball|large|creeper|burst}
  • effect:{trail|twinkle}[,{trail|twinkle}]
