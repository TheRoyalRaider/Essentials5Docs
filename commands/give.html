/ESS5/Commands/

give

Gives a select player(s) an item(s)
Usage: /give {player} {item|item:damage value} [amount [enchant:level...]]
Gives select player(s) an amount of a specified item.
    Item Metas:
  • {enchant}:{level} - Enchants a given item
  • name:{item name} - Names a given item
  • lore:{item description} - Sets a description for a given item
  • player:{player name} - Setting item skulls to player heads
    Book Metas:
  • book:{book.txt chapter} - Used on books for custom content
  • author:{book author} - Sets the author of a book
  • title:{book title} - Sets a book's title
    Firework Metas:
  • power:{firework power} - Sets a firework's power
  • color:{color[,color...]} - Changes the main color of a firework
  • fade:{color[,color...]}] - Changes the fade color of a firework
  • shape:{star|ball|large|creeper|burst} - Changes the shape of a firework
  • effect:{trail|twinkle}] - Sets the effect of a firework
