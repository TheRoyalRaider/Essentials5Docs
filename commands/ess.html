/ESS5/Commands/

ess

Returns Essentials version, and reloads config
Usage: /ess [reload|debug] OR /ess reset {player}
Lets you trigger some internal essentials functions:
  • Lists your the essentials version and overridden commands
  • Reloads your Essentials.jar configuration
  • Toggles debug mode
  • Resets a players userdata file
